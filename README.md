# Global CO2 Emissions: Trend Analysis & Industrial Drivers

## Project Overview
This project performs a data-driven analysis of historical CO2 emission datasets. It identifies key industrial sectors and geographic regions contributing to global carbon output, providing a quantitative perspective on environmental impact.

## Key Features
- **Temporal Trend Analysis:** Visualizing decadal growth rates in global carbon output.
- **Sectoral Comparison:** Quantitative ranking of industrial drivers (Power, Transport, Manufacturing).
- **Data Pre-processing:** Implementation of cleaning and normalization techniques for multi-variable environmental datasets.
